IPL API Documentation

Overview

Welcome to the IPL API documentation. This documentation provides details on how to use the API to access IPL data.

Authentication

The IPL API requires authentication using API keys. To authenticate, include your API key in the request headers.

Authorization: Bearer your-api-key

Endpoints

Here are the available endpoints:

  • /api/teams: Retrieves information about IPL teams.
  • /api/teamvteam?team1={team1}&team2={team2}: Compares the performance of two teams.
  • /api/team-record?team={team}: Retrieves analysis data for a specific team.
  • /api/batting-record?batsman={batsman}: Retrieves batting records for a specific batsman.
  • /api/bowling-record?bowler={bowler}: Retrieves bowling records for a specific bowler.

Usage

To use the API, make HTTP requests to the relevant endpoints using appropriate query parameters.

For example:

GET /api/teams

This request retrieves information about all IPL teams.
